# OOP Exercises in Java

This project is a collection of Object-Oriented Programming (OOP) exercises written in Java. It focuses on building a solid understanding of core OOP concepts through practice.

## Concepts Covered
- Classes and Objects
- Inheritance (User → Student, Instructor)
- Encapsulation (private fields, getters/setters)
- Lists and Collections (ArrayList)
- Basic relationships between objects (Course ↔ Student ↔ Instructor)

## Structure
- `Users.java` → Base class for shared user properties
- `Student.java` → Student entity with grades and GPA logic
- `Instructor.java` → Instructor entity
- `Course.java` → Handles course details and student enrollment
- `Main.java` → Runs and tests the system

## Purpose
To practice building real-world style relationships between objects and strengthen Java OOP fundamentals in preparation for backend development work.

## How to Run
```bash
javac *.java
java Main
